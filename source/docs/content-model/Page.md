This is general or dedicated page content model, can be used to handle uncategorized page.

**find it at :**
- https://www.incentro.com
- https://www.incentro.com/en-NL/services
- https://www.incentro.com/en-NL/about-us

**The data requirement :**
- Internal Name ==Short text==
- Slug ==Short text==
- Published At ==Date & time==
- SEO Title ==Short text==
- SEO Description ==Short text==
- SEO Keywords ==Short text, list==
- Hide page from search engines ==Boolean==
- Exclude links from search rankings ==Boolean==
- Title ==Short text, list==
- Navigation Title ==Short text, list== *only general page have this property*
- Subtitle ==Short text, list==
- Parent Page ==Reference== *only general page have this property*
- Hero Button ==Reference== *only general page have this property*
- Hero Highlighted Entry ==Reference== *only general page have this property*
- Hero Variant ==Short text== *only general page have this property*
- Hero Images ==Media, many files== *only general page have this property*
- Lead Text ==Short text== *only general page have this property*
- Sections ==References, many== *only general page have this property*
