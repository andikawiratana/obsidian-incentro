**The data requirement :**
- Internal Name ==Short text==
- Slug ==Short text==
- Published At ==Short text==
- SEO Title ==Short text==
- SEO Description ==Short text==
- SEO Keywords ==Short text, list==
- Hide page from search engines ==Boolean==
- Exclude links from search rankings ==Boolean==
- Title ==Short text== *general page accept list too*
- Author ==Reference== *general page doesn't have this property*
- Download File ==Media== *general page doesn't have this property*
- Download Hubspot Form Id ==Short text== *general page doesn't have this property*
- Introduction ==Long text== *general page doesn't have this property*
- Content ==Rich text== *general page doesn't have this property*
- Services ==Short text, list== *general page doesn't have this property*
- Industries ==Short text, list== *general page doesn't have this property*
- Technologies ==Short text, list== *general page doesn't have this property*
- Hero image ==Media== *general page doesn't have this property*
- Lead Text ==Short text== *general page doesn't have this property*
- Contact Person ==Reference== *general page doesn't have this property*
